## 🌟 本アプリケーションの特徴

本システムは、野球の打撃成績を「素早く」「直感的」かつ「詳細」に記録・分析するために開発された、アマチュア・大学野球向けの打撃管理アプリケーションです。

### 1. 直感的な投球コース自由プロットUI
従来の9分割や25分割といった大まかな「コース選択式」とは異なり、キャッチャー目線の配球チャート画像上を**直接クリック（タップ）して正確な球跡をプロットできる自由選択方式**を採用しています。これにより、インコース高め、アウトロー低めのボール球といったシビアなコース位置を視覚的かつ正確にデータ化できます。

### 2. カタカナ・別称に対応した「ハイブリッド音声入力」
現場での手軽な入力を実現するため、`streamlit-mic-recorder` を用いた音声認識入力を搭載しています。
「ストレート ヒット ショート 2打点」のように連続して喋るだけで、内部の辞書マッピング処理により、「ショート ➔ 遊」「ピッチャー ➔ 投」「サード ➔ 三」のように、**一般的な野球の呼び方（カタカナ・別称）を公式記録用の漢字一文字へと自動変換**してフォームに反映します。また、ホームランの発話時は自動で打点「1」を補完するスマートロジックも組み込まれています。

### 3. 三振・四死球時の入力自動スキップ（安全ガード）
打撃結果で「三振」「四球」「死球」が選択された場合、打球ポジションの入力は不要（または「なし」）となるため、**システムが自動でポジション入力をスキップする制御**を行います。これにより、入力の手間を減らすと同時に、無矛盾な正しいデータのみをデータベースに保存する仕組みにしています。

### 4. リアルタイムな通算＆試合ごとスタッツ分析
一時保存データおよびSQLite3データベースに永続化された過去の全打席データから、打率・長打率・OPSなどの基本スタッツをはじめ、**「ヒットした球種の割合」や「打球方向の傾向」を自動計算**し、即座にデータフレーム形式で出力します。

## 🚀 起動方法

本アプリケーションはPythonのWebフレームワーク「Streamlit」を利用して動作します。ローカル環境で起動する手順は以下の通りです。

### 1. 必要なライブラリのインストール（初回のみ）
ターミナル（またはコマンドプロンプト）を開き、以下のコマンドを実行して必要な外部パッケージを一括インストールします。
```bash
pip install streamlit pandas pillow streamlit-mic-recorder streamlit-image-coordinates openpyxl
